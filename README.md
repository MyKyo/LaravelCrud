# Laravel CRUD App

![Laravel Logo](https://laravel.com/img/logotype.min.svg)

A simple CRUD application built with Laravel.

## Features

- **Create**: Add new records
- **Read**: View records
- **Update**: Edit records
- **Delete**: Remove records
